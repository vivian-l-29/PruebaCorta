HTML

HyperText Markup Language

Lenguaje estándar utilizado para crear y estructurar páginas web. Está compuesto por una serie de etiquetas que definen diferentes partes del contenido de una página, como encabezados, párrafos, imágenes, enlaces y otros elementos.

CSS

Cascading Style Sheets

Lenguaje de estilo utilizado para definir la apariencia y el diseño de una página web escrita en HTML. Permite separar el contenido (HTML) de la presentación visual, lo que facilita la gestión y el mantenimiento de un sitio web.
